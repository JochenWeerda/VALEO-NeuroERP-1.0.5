ERP Server Manager
=================

Der ERP Server Manager ist ein Tool zur Verwaltung der Backend- und Frontend-Server des AI-Driven ERP Systems.

Funktionen:
----------
- Starten und Stoppen der Server
- Pausieren und Fortsetzen der Server
- Automatische Pausierung nach 30 Minuten Inaktivität
- Automatisches Herunterfahren nach 60 Minuten Inaktivität
- Status-Anzeige für beide Server
- Log-Datei für Fehler und Ereignisse

Installation:
------------
1. Führen Sie die Setup-Datei aus
2. Folgen Sie dem Installationsassistenten
3. Geben Sie die korrekten Pfade zu Ihren Backend- und Frontend-Verzeichnissen ein
4. Wählen Sie die gewünschten Installationsoptionen

Verwendung:
----------
- Starten Sie den Server Manager über das Desktop-Symbol oder das Startmenü
- Verwenden Sie die Buttons zum Steuern der Server
- Die Status-Ampel zeigt den aktuellen Zustand der Server an
- Die Log-Datei finden Sie im Installationsverzeichnis unter "logs\server_manager.log"

Systemanforderungen:
------------------
- Windows 10 oder höher
- PowerShell 5.1 oder höher
- Python 3.8 oder höher (für Backend)
- Node.js 14 oder höher (für Frontend)

Support:
-------
Bei Problemen oder Fragen wenden Sie sich bitte an den Systemadministrator.
